# SafetyNet SNet.jar extractor

Command line tool to download and extract the latest version of the Google Play Services SafetyNet Jar file

To use compile and run app. The latest version of SNet will be downloaded to current directory, along with the manifest metadata

```
Downloading the latest SNet...
Manifest info:
{
  "percent": 100,
  "url": "https://www.gstatic.com/android/snet"
}


Success! Downloaded SafetyNet file saved as 'snet.jar'
```
